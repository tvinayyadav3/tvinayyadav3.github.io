Promise

It is an object that represents the eventual completion or failure of an asynchronous operation.It is used for andling asynchronous operations, such as API calls or reading files, in more organized and readale way.

A JS promise object can be

    • Pending
      FulFilled
      Rejected

    • Syntax: let promise =new promise(function(resolve,reject))

    eg: const promise=new Promise(function(resolve,reject){ var x='javascript' var y='javascript1' if(x==y) { return resolve() }else{ return reject() } }) promise.then((x)=>document.write("Successful")) .catch((err)=>document.write("Error"))
